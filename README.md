# FASU Hacker Dashboard

Advanced Android security dashboard for Termux.

## Features

- ADB device control
- Network scanning
- Battery monitor
- Wi-Fi scanner
- Automation tools
- Kali NetHunter launcher
- Screenshot tools
- System information

---

## Installation

### Install Git

```bash
pkg install git -y
```

### Clone Repository

```bash
git clone YOUR_REPOSITORY_LINK
```

### Open Folder

```bash
cd hacker-dashboard
```

### Give Permission

```bash
chmod +x dashboard.sh
```

### Run Dashboard

```bash
./dashboard.sh
```

---

## Requirements

- Termux
- Android device
- ADB enabled
- Optional: Kali NetHunter
