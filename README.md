## Installation

### Install Git

```bash
pkg install git -y
```

### Clone Repository

```bash
git clone https://github.com/Fasu2007/fasu_android_hacking.git
```

### Open Folder

```bash
cd hacker-dashboard
```

### Give Permission

```bash
chmod +x dashboard.sh
```

### Run Dashboard

```bash
./dashboard.sh
```
